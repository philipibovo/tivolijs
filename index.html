O que é tivolijs?

É uma biblioteca CSS/JQuery criada para facilitar o uso de formulários em páginas HTML, deixando os elementos do formulário mais elegantes e criando facilidades para o desenvolvedor e para o usuário.

Como usar?

1° faça o download do pacote.

Acesse o repositório do projeto no Github:

https://github.com/philipibovo/tivolijs

Clone o projeto do Github diretamente em seu diretório usando o terminal:

Terminal

git clone https://github.com/philipibovo/tivolijs.git

Faça o download do pacote:

OU

2° Adicione o CSS em sua página HTML.

HTML

<link rel="stylesheet" href="css/tivoli-0.0.1.min.css">

3° Adicione o Javascript em sua página HTML.

HTML

<script type="text/javascript" src="js/tivoli-0.0.1.min.js>

4° Pronto! Sua página já está pronta pra usar o tivolijs.

Aplicando estilo

É possível aplicar o estilo de duas formas diferentes:

- No <form>, aplicando a classe 'tivoli-style', assim todos os campos e botões presentes dentro do formulário. Neste caso, não é necessário aplicar classes de estilo no campos presentes dentro do formulário ficarão estilizados no padrão tivolijs.

HTML

<form id="myForm" class="tivoli-style" method="post" action="result.html" >

<label for="myText">My Label:</label>

<input type="text" id="myText" value="" />

<label for="mySelect">My Select:</label>

<select id="mySelect">

<option value="1">Option 1</option>

<option value="2">Option 2</option>

</select>

<button>My Button</button>

</form>

- Diretamente no elemento, aplicando as classes: 'tivoli-label' nos elementos <label>, 'tivoli-field' nos elementos <input type="text">, <input type="password">, <input type="date">, <textarea>, <select>... e 'tivoli-btn' nos elementos <button>, <input type="button">, <input type="submit">...

HTML

<!-- label -->

<label class="tivoli-label" for="idFormDestiny">My Label:</label>

 

<!-- input text -->

<input type="text" id="myText" class="tivoli-field" value="" />

 

<!-- select -->

<select id="mySelect" class="tivoli-field">

    <option value="1">Option 1</option>

    <option value="2">Option 2</option>

</select>

 

<!-- button -->

<button class="tivoli-btn">My Button</button>

Mensagem informativa

É possível exibir uma mensagem informativa embaixo de um campo. Para isso, adicione logo abaixo do campo, um elemento do tipo <p> com a classe 'tivoli-msg'.

Mensagem

HTML

<label class="tivoli-label" for="example1">Nome:</label>

<input type="text" id="example1" class="tivoli-field" value="" />

<p class="tivoli-msg">Mensagem</p>

É possível utilizar 5 cores diferentes, a cor padrão é a cinza (class="primary"), ou seja, se não adicionar a classe referente a cor, ele adotará a cor cinza (class="primary").

Controle de Tamanho

Block: Diferente dos campos, os botões não tem largura de 100% do elemento pai. Para que o botão ocupe 100% do elemento pai, adicione a classe 'block'. Esta classe pode ser usada em todo tipo de botão.

HTML

<button id="exampleButtonBlock" class="tivoli-btn block">Button Block</button>

Tamanho Pequeno: Para usar o botão no tamanho pequeno adicione a classe 'small'. Esta classe pode ser usada em todo tipo de botão.

HTML

<button id="exampleButtonSmall" class="tivoli-btn small">Button Small</button>

Tamanho Grande: Para usar o botão no tamanho grande adicione a classe 'big'. Esta classe pode ser usada em todo tipo de botão.

HTML

<button id="exampleButtonBig" class="tivoli-btn big">Button Big</button>

Validação

Com o tivoljs também é possível fazer a validação de um formulário de forma fácil e rápida.

A validação do formulário é feita no submit do form. Caso a validação falhe, o submit é abortado.

Para que um <form> seja validado, é necessário incluir a classe 'tivoli-validate' ao form.

HTML

<form id="myForm" class="tivoli-validate" method="post" action="myaction.html">

...

</form>

Todo campo de preenchimento obrigatório a ser validado, deve conter o atributo 'required'.

<input type="text" id="myFormText" value="" required />

Quando a validação antes do submit falha, o submit é abortado e os campos inválidos serão destacados em vermelho, enquanto os outros campos com 'required' que estão corretos serão destacados em verde.

É possível exibir uma mensagem de erro abaixo do campo, caso o mesmo seja inválido. Para isso inclua o atributo data-tivoli-error="Conteúdo da Mensagem" no campo.

HTML

<label class="tivoli-label error" for="myFormName">Nome:</label>

<input type="text" id="myFormName" value="" class="tivoli-field error" data-tivoli-error="Preencha o nome" required />
